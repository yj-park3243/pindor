## 2. 매칭 알고리즘 구현
- [ ] 매칭 큐 워커 (BullMQ) — 10초마다 핀별 매칭 풀 스캔
- [ ] 같은 핀 + 같은 종목에서 MatchScore 최고 쌍 자동 매칭
- [ ] 대기 시간에 따른 MMR 범위 점진적 확대 (0~2분: ±100, 5분: ±200, 10분+: ±500)
- [ ] 매칭 성사 시 양측에 푸시 알림 + 채팅방 자동 생성
- [ ] 매칭 수락/거절 타임아웃 (5분)

> 설계 문서: [design-matching-algorithm.md](./design-matching-algorithm.md)

## 3. 게시판 입장 조건 — 1회 이상 매칭 참가

  - 조회는 누구나 가능
- [ ] 서버: `POST /pins/:pinId/posts`, `POST .../comments`, `POST .../like` 라우트에 미들웨어 적용
- [ ] 앱: 게시글 작성 시 권한 없으면 "이 핀에서 1회 이상 매칭에 참가해야 합니다" 안내 다이얼로그
