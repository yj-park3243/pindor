### 13. 매칭 목록 (탭 3)
- **역할**: 내 매칭 전체 보기
- **요소**: 앱바 "매칭" + 세그먼트 탭(진행중/완료/취소) + 매칭 카드들
- **매칭 카드 구성**: 좌측 색상 바(상태별) + 상대 아바타(60px) + 닉네임+점수뱃지 + 상태칩 + 종목+날짜 + 핀위치/시간/만남횟수(PENDING_ACCEPT일 때)
- **진행중 탭 상단**: 매칭 슬롯 카운터("매칭 가능: N개 남음")

### 14. 매칭 생성
- **역할**: 매칭 요청 생성 폼
- **요소 (스텝 카드 형태)**:
  1. 핀 정보 (읽기전용, 오렌지 배경)
  2. 종목 선택 (Wrap 칩)
  3. 매칭 모드 (랭크/친선 토글, ON=랭크)
  4. 날짜 선택 (당일/내일만, 캘린더 피커)
  5. 시간대 선택 (4열 그리드 8개 + 하단 "하루종일" 풀사이즈, 당일 지난 시간 비활성화)
  6. 성별 조건 (상관없음/같은 성별만 세그먼트)
  7. 나이차 제한 (친선 모드일 때만 표시): 슬라이더 0~7, 0=제한없음, 1~7=해당 나이차 이내만 매칭. 라벨 "나이차 N살 이내" (0일 때 "제한 없음")
  8. "매칭 요청하기" 버튼
- **조건부 표시**: 랭크 매칭일 때는 나이차 슬라이더 숨김 (실력 기반 매칭이므로), 친선일 때만 노출

### 15. 매칭 수락

- **뒤로가기 차단 (전체화면 잠금)**
- **요소 (위→아래)**:
  1. "매칭이 잡혔습니다!" 텍스트(22pt Bold 오렌지) + 스포츠 아이콘
  2. 상대 프로필 이미지(88px 원형, 오렌지 테두리)
  3. 닉네임(24pt Bold) + 프로필 메시지(이탤릭)
  4. 점수+경기수 + 만남횟수("처음 만나는 상대" / "N번 만난 상대")
  5. 매칭 정보 카드: 핀위치 + 종목태그 + 랭크/친선태그 + 날짜·시간
  6. 카운트다운 타이머(28pt Bold, 색상 변화: 초록→노랑→빨강) + 프로그레스 바
  7. 거절(-15점) / 수락 2버튼 (수락 후 → "상대 응답 대기중..." 로딩)

### 16. 매칭 상세

- **요소**:

  - 친선 배너 (있을 때, 오렌지)
  - 상태 배너 (수락대기/경기확정/완료 등)
  - 매칭 정보 카드: 종목 / 핀 / 경기일시 / 확정날짜·시간 / 장소
  - 버튼들: 채팅하기(파랑) / 승부 결과 입력(초록, 바텀시트) / 의의 제기(오렌지)
  - 햄버거 메뉴: 경기 포기 / 노쇼 신고

### 17. 승부 결과 입력 (바텀시트, 매칭 상세+채팅방 공통)
- **역할**: 승/패/무 선택 + 증빙 사진 + 매너 점수
- **요소**:
  1. "승부 결과 입력" 제목 + "vs 닉네임"
  2. 승리(파랑)/무승부(회색)/패배(빨강) 3버튼 (아이콘+텍스트, 선택 시 테두리+배경)
  3. 사진 첨부 (필수): 카메라 촬영만, 최대 2장, 썸네일 미리보기+X삭제
  4. 매너 점수: 별 1~5개 (노란 별)
  5. "결과 제출" 버튼 (사진 없으면 비활성화)
