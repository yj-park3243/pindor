# 핀돌 (PINDOR) — 피그마 디자인 프롬프트 v2

## 앱 개요
- **이름**: 핀돌 (PINDOR)
- **설명**: 위치 기반 스포츠 1:1 매칭 + 크루(팀) 매칭 플랫폼
- **플랫폼**: iOS / Android (Flutter)
- **테마**: 다크 모드 전용

- **하단 탭바**: 4개 탭 (홈/핀/매칭/마이), 배경 #1A1A1A, 선택 시 오렌지
- 크루 기능은 별도 탭 없이, 핀 지도(탭 2) 우측 상단 **모드 토글**로 진입
  - 토글: `개인` ↔ `크루` (캡슐형 세그먼트, 선택 시 오렌지 배경)
  - **개인 모드**: 개인 종목 마커 + 1:1 매칭 중심
  - **크루 모드**: 크루 종목 마커 + 크루 목록/크루 매칭 중심

## 화면 목록 (64개)

---

### 1. 스플래시
- **경로**: 앱 시작 시 최초 화면
- **역할**: 로고 표시 + 인증 상태 확인 후 자동 이동
- **요소**: 핀돌 로고 (중앙), 페이드아웃 애니메이션
- **이동**: 로그인 안됨→온보딩, 로그인됨→홈

### 2. 온보딩 (3페이지)
- **역할**: 앱 소개 슬라이드 3장
- **요소**: 페이지뷰 + 도트 인디케이터 + 건너뛰기/다음/시작 버튼
- **페이지별 내용**:
  1. "내 근처에서 대결 상대를 찾아보세요" (위치 아이콘, 오렌지 배경)
  2. "실력 기반 공정한 매칭" (ELO 아이콘, 핑크 배경)
  3. "랭킹으로 실력을 증명하세요" (트로피 아이콘, 골드 배경)

### 3. 로그인
- **역할**: 소셜 로그인 진입점
- **요소**: "P" 로고 원형 + "핀돌" 텍스트 + "내 근처 스포츠 대결 매칭" + Apple 로그인 버튼(검정) + Google 로그인 버튼(아웃라인) + 약관 링크

### 4. 프로필 설정 (1/3단계)
- **역할**: 닉네임 + 프로필 사진 설정
- **요소**: 진행 바(1/3) + 프로필 이미지 피커(100px 원형, 카메라 아이콘) + 닉네임 입력 + 랜덤 생성 버튼(주사위 아이콘) + 다음 버튼

### 5. 스포츠 프로필 설정 (2/3단계)
- **역할**: 주 종목 선택 + 골프 핸디캡
- **요소**: 진행 바(2/3) + 종목 그리드(3열, 8종목 아이콘) + G-핸디캡 슬라이더(0~54) + 다음 버튼

### 6. 위치 설정 (3/3단계)
- **역할**: 자주 가는 핀(위치) 선택
- **요소**: 진행 바(3/3) + 네이버 지도(전체 화면) + 핀 마커들 + GPS 현위치 버튼 + 핀 선택 확인 버튼

### 7. 홈 (탭 1)
- **역할**: 메인 피드
- **요소 (위→아래)**:
  1. 앱바: "핀돌" 로고(오렌지) + 알림 벨 아이콘(빨간 뱃지)
  2. 매칭 수락 대기 배너 (있을 때만, 파란 그라데이션, "매칭 수락 대기중!")
  3. 고정 공지 배너 (있을 때만, 노란 배경, 메가폰 아이콘)
  4. "오늘 대결 나가고 싶다!" CTA (오렌지 그라데이션, 번개 아이콘)
  5. 내 핀 상태 카드: 핀명+종목 / 순위·점수·활동중 3열 / 전적 태그+배치중 태그+게시판 버튼
  6. 내 크루 현황 카드 (크루 가입 시 표시): 크루명+종목+핀 / 크루 점수+티어 / "크루 홈" 버튼. 여러 크루 가입 시 가로 스크롤 카드
  7. 최근 전적 섹션: 최대 5개, 승(파랑)/패(빨강)/무(회색) 뱃지 + 상대 아바타 + 닉네임 + 종목·날짜
  8. 게시판 최신글: 최대 3개, 종목 태그 + 제목 + 작성자·시간 + 댓글 수

### 8. 즉시 매칭
- **역할**: 오늘 대결 빠른 매칭
- **요소**: 핀 선택 칩(가로 스크롤) + 종목 탭 + 시간대 선택(3시간 단위 8개 + 하루종일) + 생성 버튼

---

### 9. 핀 지도 (탭 2)
- **역할**: 전체 핀을 지도에 표시 + 개인/크루 모드 전환
- **요소**:
  - 네이버 지도(전체) + 종목별 마커(색상 구분) + GPS 현위치 버튼
  - **우측 상단**: 모드 토글 (캡슐형 세그먼트 `개인` | `크루`)
  - 핀 탭 시 → 모드에 따라 다른 상세 시트 표시
- **개인 모드** (기본):
  - 마커: 개인 종목 기준 (골프·당구·테니스 등 개인 매칭 가능 종목)
  - 핀 탭 → 10-A. 개인 핀 상세 시트
- **크루 모드**:
  - 마커: 크루 종목 기준 (축구·농구·야구 등 크루 매칭 종목) + 크루 수 뱃지
  - 핀 탭 → 10-B. 크루 핀 상세 시트
- **모드 전환 시**: 마커 애니메이션 전환 (fade), 종목 필터 칩도 해당 모드 종목으로 변경

### 10-A. 핀 상세 시트 — 개인 모드 (바텀시트)
- **역할**: 핀 선택 시 개인 매칭 정보
- **요소**: 드래그 핸들 + 핀명(20pt Bold)+활동인원 + 종목 선택 칩(가로) + "랭크 매칭"/"친선 매칭" 2버튼 + "게시판"/"전체 랭킹" 2버튼 + TOP 10 개인 랭킹 리스트

### 10-B. 핀 상세 시트 — 크루 모드 (바텀시트)
- **역할**: 핀 선택 시 크루 목록 + 크루 매칭 정보
- **요소**:
  - 드래그 핸들 + 핀명(20pt Bold) + 활동 크루 수
  - 종목 선택 칩(가로, 크루 종목만)
  - **크루 목록** (카드형, 스크롤): 크루 이미지(48px) + 크루명 + 종목 태그 + 크루 점수/티어 + 멤버 수 + 모집중 뱃지(초록)
  - "크루 만들기" 버튼 (하단 고정, 오렌지 아웃라인)
  - 크루 카드 탭 → 크루 홈(34번) 이동
  - TOP 5 크루 랭킹 미니 리스트

### 11. 핀 랭킹
- **역할**: 특정 핀의 전체 랭킹
- **요소**: 앱바(핀명) + 내 랭킹 카드(있을 때) + TOP 3 포디움(금/은/동 메달) + 4위 이하 리스트(순위·아바타·닉네임·점수·티어뱃지)

### 12. 내 랭킹
- **역할**: 나의 핀별 랭킹 조회
- **요소**: 핀 선택 칩(가로) + 내 순위 카드 + 점수 추이 그래프 + 해당 핀 상위 랭킹 리스트

---

### 13. 매칭 목록 (탭 3)
- **역할**: 내 매칭 전체 보기
- **요소**: 앱바 "매칭" + 세그먼트 탭(진행중/완료/취소) + 매칭 카드들
- **매칭 카드 구성**: 좌측 색상 바(상태별) + 상대 아바타(60px) + 닉네임+점수뱃지 + 상태칩 + 종목+날짜 + 핀위치/시간/만남횟수(PENDING_ACCEPT일 때)
- **진행중 탭 상단**: 매칭 슬롯 카운터("매칭 가능: N개 남음")

### 14. 매칭 생성
- **역할**: 매칭 요청 생성 폼
- **요소 (스텝 카드 형태)**:
  1. 핀 정보 (읽기전용, 오렌지 배경)
  2. 종목 선택 (Wrap 칩)
  3. 매칭 모드 (랭크/친선 토글, ON=랭크)
  4. 날짜 선택 (당일/내일만, 캘린더 피커)
  5. 시간대 선택 (4열 그리드 8개 + 하단 "하루종일" 풀사이즈, 당일 지난 시간 비활성화)
  6. 성별 조건 (상관없음/같은 성별만 세그먼트)
  7. 나이차 제한 (친선 모드일 때만 표시): 슬라이더 0~7, 0=제한없음, 1~7=해당 나이차 이내만 매칭. 라벨 "나이차 N살 이내" (0일 때 "제한 없음")
  8. "매칭 요청하기" 버튼
- **조건부 표시**: 랭크 매칭일 때는 나이차 슬라이더 숨김 (실력 기반 매칭이므로), 친선일 때만 노출

### 15. 매칭 수락
- **역할**: 상대 찾음 → 10분 내 수락/거절
- **뒤로가기 차단 (전체화면 잠금)**
- **요소 (위→아래)**:
  1. "매칭이 잡혔습니다!" 텍스트(22pt Bold 오렌지) + 스포츠 아이콘
  2. 상대 프로필 이미지(88px 원형, 오렌지 테두리)
  3. 닉네임(24pt Bold) + 프로필 메시지(이탤릭)
  4. 점수+경기수 + 만남횟수("처음 만나는 상대" / "N번 만난 상대")
  5. 매칭 정보 카드: 핀위치 + 종목태그 + 랭크/친선태그 + 날짜·시간
  6. 카운트다운 타이머(28pt Bold, 색상 변화: 초록→노랑→빨강) + 프로그레스 바
  7. 거절(-15점) / 수락 2버튼 (수락 후 → "상대 응답 대기중..." 로딩)

### 16. 매칭 상세
- **역할**: 매칭 정보 + 액션
- **요소**:
  - VS 헤더 (오렌지 그라데이션): 내 아바타 + "나" / VS / 상대 아바타+닉네임
  - 친선 배너 (있을 때, 오렌지)
  - 상태 배너 (수락대기/경기확정/완료 등)
  - 매칭 정보 카드: 종목 / 핀 / 경기일시 / 확정날짜·시간 / 장소
  - 버튼들: 채팅하기(파랑) / 승부 결과 입력(초록, 바텀시트) / 의의 제기(오렌지)
  - 햄버거 메뉴: 경기 포기 / 노쇼 신고

### 17. 승부 결과 입력 (바텀시트, 매칭 상세+채팅방 공통)
- **역할**: 승/패/무 선택 + 증빙 사진 + 매너 점수
- **요소**:
  1. "승부 결과 입력" 제목 + "vs 닉네임"
  2. 승리(파랑)/무승부(회색)/패배(빨강) 3버튼 (아이콘+텍스트, 선택 시 테두리+배경)
  3. 사진 첨부 (필수): 카메라 촬영만, 최대 2장, 썸네일 미리보기+X삭제
  4. 매너 점수: 별 1~5개 (노란 별)
  5. "결과 제출" 버튼 (사진 없으면 비활성화)

---

### 18. 채팅 목록
- **역할**: 활성 채팅방 리스트 (1:1 개인 + 크루 그룹)
- **요소**: 세그먼트 탭("개인"/"크루") + 채팅방 타일(아바타 + 이름 + 마지막 메시지 미리보기 + 시간 + 안읽음 뱃지)
- **크루 채팅방 타일 구분**: 크루 엠블럼 이미지(64px, 라운드 12px) + 크루명 + "[크루]" 배지(오렌지 소형 칩) + 마지막 메시지
- **이동**: 개인 채팅방 탭 → 채팅방(19번), 크루 채팅방 탭 → 크루 채팅방(45번)

### 19. 채팅방
- **역할**: 1:1 메시지 + 경기 관련 액션
- **요소**: 앱바(상대 닉네임) + 메시지 목록(내 메시지=오렌지/오른쪽, 상대=회색/왼쪽) + 타이핑 인디케이터 + "승부 결과 입력" 버튼(입력바 바로 위, 초록 풀사이즈) + 채팅 입력바(텍스트+이미지 첨부+전송)

---

### 20. 경기 결과 확인
- **역할**: 상대방 결과 확인 + 인증/이의
- **요소**: 상태 배너 + 내 결과 표시 + 상대 결과 표시 + "결과 인증"(초록) / "이의 신청"(빨강) 버튼

### 21. 점수 변동 결과
- **역할**: 경기 후 점수 변동 애니메이션
- **요소**: "승리!"/"패배" 타이틀(애니메이션) + 점수 카운터 애니메이션(이전→신규) + 점수 변동(+25/-15) + 랭크 변동(있을 때) + 배치 진행률(배치 중일 때) + 승리 시 컨페티

---

### 22. 내 프로필 (탭 4 → 마이)
- **역할**: 프로필 허브
- **요소**: 프로필 헤더(아바타+닉네임+점수+티어뱃지) + 스포츠 프로필 카드(종목·점수·경기수) + 종목 추가 버튼 + 메뉴(활동설정: 자주가는핀/종목변경, 계정: 알림/문의/설정/로그아웃)

### 23. 프로필 수정
- **요소**: 아바타 피커 + 닉네임 입력 + 저장 버튼

### 24. 스포츠 프로필 관리
- **요소**: 종목별 카드(종목·점수·티어·전적·매칭 문구) + 매칭 문구 수정 바텀시트

### 25. 알림 목록

### 26. 알림 설정

- **크루 알림 항목 (v2 추가)**: 크루 가입 신청 / 가입 승인·거절 / 크루 매칭 신청 / 크루 공지 / 크루원 추방

### 27. 설정

## 크루(CREW) 화면 (33~52번)

> 크루: 종목 + 핀(위치)을 기반으로 구성된 스포츠 팀 단위. 개인은 여러 크루에 동시 가입 가능.
> 역할 체계: 크루장 > 부크루장 > 크루원
> 크루 전용 ELO 점수는 개인 점수와 별도로 관리됨.

---

### 33. 내 크루 목록
- **역할**: 내가 가입한 크루 전체 목록 + 크루 탐색 진입
- **진입 경로**: 마이 탭(22번) 메뉴 "내 크루" / 홈(7번) 크루 현황 카드 "전체보기"
- **요소 (위→아래)**:
  1. 앱바: "내 크루" 텍스트(20pt Bold) + "크루 만들기" 아이콘 버튼(상단 우측, + 아이콘)
  2. 크루 가입 없는 경우: 빈 상태 (방패 아이콘 64px + "아직 가입한 크루가 없어요" + "크루를 만들거나 탐색해보세요" + "핀 지도에서 크루 탐색하기" 버튼(오렌지))
  3. 크루 있는 경우:
     - "내 크루" 섹션 헤더 + 전체 N개 카운터
     - 크루 카드 리스트 (세로 스크롤):
       - 크루 엠블럼(56px 라운드) + 크루명(16pt SemiBold) + 종목 칩 + 핀명 칩
       - 크루 점수(ELO) + 티어 뱃지 + 크루원 수(N/최대)
       - 내 역할 뱃지: "크루장"(오렌지) / "부크루장"(노랑) / "크루원"(회색)
       - 모집중 뱃지(초록, 모집중일 때만)
       - 미읽은 채팅 뱃지(빨간 원형 숫자)
     - 카드 탭 → 크루 홈(34번)

---

### 34. 크루 홈
- **역할**: 개별 크루의 메인 페이지
- **요소**:
  1. 앱바: 뒤로가기 + 크루명(18pt Bold) + 더보기(···) 아이콘
     - 더보기 메뉴(크루원): 크루 탈퇴
     - 더보기 메뉴(크루장/부크루장): 크루 관리 + 크루 탈퇴(크루장은 양도 후 탈퇴)
  2. 크루 헤더 카드 (오렌지 그라데이션 상단):
     - 크루 엠블럼 이미지(80px 원형, 탭 시 이미지 확대) + "편집" 아이콘(크루장만)
     - 크루명(22pt Bold) + 종목 칩 + 핀명 칩
     - 소개글(14pt, 최대 2줄 + 더보기)
     - 창단일 ("YYYY.MM.DD 창단") + 크루원 수 ("N / 최대M명")
     - 크루 점수(ELO 숫자, 24pt Bold) + 티어 뱃지 + 전체 전적(N승 N패)
     - 모집 상태 뱃지: "모집중"(초록) / "모집마감"(회색)
  3. 크루원 미리보기 (가로 스크롤, 최대 6명):
     - 아바타(44px 원형) + 닉네임(10pt) + 역할 소형 뱃지
     - 마지막 셀: "전체보기 >" → 크루원 목록(38번)
  4. 탭 바: 경기 / 공지 / 채팅 3개 탭
     - 경기 탭: 크루 경기 목록 미리보기 (최근 3건 + "전체보기") → 크루 경기 목록(40번)
     - 공지 탭: 크루 공지 목록 (최근 3개 + FAB) → 크루 공지(47번)
     - 채팅 탭: 크루 채팅방 바로 진입 → 크루 채팅방(45번)
  5. 하단 고정 버튼:
     - 크루장/부크루장: "크루 경기 신청" 버튼(오렌지, 풀사이즈)
     - 크루원: "채팅방 입장" 버튼(파랑, 풀사이즈)

---

### 35. 크루 만들기
- **역할**: 새 크루 생성 폼
- **요소 (스텝 형태, 단일 스크롤)**:
  1. 앱바: 뒤로가기 + "크루 만들기"
  2. 크루 엠블럼 설정:
     - 원형 이미지 피커(80px, 기본 이미지 제공 5종) + 카메라/갤러리 선택
  3. 크루명 입력 (최대 20자, 중복 체크 실시간)
  4. 소개글 입력 (최대 100자, 선택 입력)
  5. 종목 선택: 그리드(3열, 전체 종목 아이콘+텍스트, 단일 선택)
  6. 핀 선택: 가로 스크롤 칩 (내 핀 목록) + "지도에서 선택" 버튼
  7. 최대 인원 안내 (자동 설정, 수정 불가):
     - 단체 종목(축구/야구/농구): "인원 제한 없음" 텍스트 표시
     - 소규모 종목(테니스/배드민턴/골프/당구/볼링): "최대 3명" 텍스트 표시
  8. 모집 여부: 토글 (ON=모집중, 기본값 ON)
  9. "크루 만들기" 버튼 (하단 고정, 크루명+종목+핀 미입력 시 비활성화)
- **완료 후**: 크루 홈(34번) 이동 + "크루가 만들어졌습니다!" 토스트

---

### 36. 크루 탐색 (핀 내 크루 목록)
- **역할**: 특정 핀에 속한 크루 목록 탐색 및 가입 신청
- **진입 경로**: 핀 지도 크루 모드(9번)에서 핀 탭 → 10-B 크루 상세 시트의 크루 카드 탭 / 내 크루 목록(33번)
- **요소**:
  1. 앱바: 뒤로가기 + "[핀명] 크루" (핀 상세에서 진입 시) / "주변 크루 탐색" (탭 홈에서 진입 시)
  2. 핀 선택 칩 (가로 스크롤, "핀 탐색" 진입 시): 내 핀 목록 칩
  3. 종목 필터 칩 (가로 스크롤): 전체 / 종목별 필터
  4. 정렬 드롭다운: 크루원 많은순 / 점수 높은순 / 최신순
  5. 크루 카드 리스트:
     - 크루 엠블럼(56px) + 크루명(16pt SemiBold) + 종목 칩 + 핀명 칩
     - 소개글 (최대 1줄 말줄임)
     - 크루 점수(ELO) + 티어 뱃지 + 크루원 수(단체: N명 / 소규모: N/3명) + 창단일
     - 모집 상태 뱃지: "모집중"(초록) / "모집마감"(회색)
     - 우측 하단 버튼: "가입 신청"(오렌지 아웃라인) / "신청중"(회색, 이미 신청) / "가입됨"(초록, 이미 가입)
  6. 빈 상태: "이 핀에 크루가 없어요" + "크루 만들기" 버튼
- **가입 신청 탭 시**: 가입 신청 확인 모달 → 신청 완료 토스트 "크루장의 승인을 기다리고 있어요"

### 37. 크루 가입 신청 관리 (바텀시트)
- **역할**: 크루장/부크루장이 가입 신청 목록을 보고 승인/거절
- **진입 경로**: 알림 → "N명이 가입을 신청했습니다" 탭 / 크루 관리(39번)의 "가입 신청 관리"
- **요소**:
  1. 바텀시트 핸들 + "가입 신청 관리" 제목 + 신청 건수 뱃지
  2. 신청자 카드 리스트:
     - 프로필 아바타(48px) + 닉네임 + 개인 ELO 점수 + 티어 뱃지
     - 종목 스포츠 프로필 유무 표시 (해당 종목 경기 수)
     - 신청 시간 (N분 전 / N시간 전)
     - "거절"(아웃라인 빨강) / "승인"(오렌지) 2버튼
  3. 빈 상태: "신청 대기중인 크루원이 없어요"

---

### 38. 크루원 목록
- **역할**: 크루 전체 크루원 조회 + 관리 액션
- **진입 경로**: 크루 홈(34번) 크루원 미리보기 "전체보기" / 크루 관리(39번)
- **요소**:
  1. 앱바: 뒤로가기 + "크루원 (N명)"
  2. 역할 필터 칩: 전체 / 크루장 / 부크루장 / 크루원
  3. 크루원 카드 리스트:
     - 프로필 아바타(48px) + 닉네임(16pt) + 역할 뱃지
     - 개인 ELO 점수 + 티어 뱃지 + 해당 종목 전적(N승 N패)
     - 가입일 (YYYY.MM.DD)
     - 크루장/부크루장인 경우: 우측 더보기(···) 버튼
  4. 더보기 액션 시트 (크루장/부크루장):
     - 크루장 메뉴: "부크루장으로 지정" / "크루원으로 강등" / "추방" / "크루장 양도"
     - 부크루장 메뉴: "추방" (자신보다 낮은 권한만 적용 가능)
  5. 크루장 양도 확인 모달: "정말로 크루장을 양도하시겠어요? 양도 후 부크루장으로 강등됩니다."
  6. 추방 확인 모달: "추방된 크루원은 재가입 신청이 가능합니다."

### 39. 크루 관리
- **역할**: 크루장/부크루장의 크루 설정 관리
- **진입 경로**: 크루 홈(34번) 앱바 더보기 → "크루 관리"
- **접근 권한**: 크루장 / 부크루장 (일부 항목은 크루장만)
- **요소**:
  1. 앱바: 뒤로가기 + "크루 관리"
  2. 크루 정보 섹션:
     - "크루 정보 수정" → 크루명, 소개글, 엠블럼 변경 (크루장만)
  3. 모집 섹션:
     - "멤버 모집" 토글: ON(초록, "모집중") / OFF(회색, "모집마감")
     - 모집 중일 때 최대 인원 표시 및 수정 가능 (크루장만)
  4. 멤버 섹션:
     - "가입 신청 관리" → 신청 건수 뱃지 (크루장/부크루장 접근 가능)
     - "크루원 목록" → 크루원 목록(38번)
  5. 크루 경기 섹션 (크루장만):
     - "경기 결과 입력 권한": "크루장만" / "크루장+부크루장" 선택
  6. 위험 섹션 (크루장만):
     - "크루 해체" (빨간 텍스트): 확인 모달 → "크루명을 입력해 확인해주세요" + 해체 버튼
     - 크루 해체 시 모든 크루원에게 푸시 알림 발송

### 40. 크루 경기 목록
- **역할**: 해당 크루의 크루 매칭 전체 이력
- **진입 경로**: 크루 홈(34번) 경기 탭
- **요소**:
  1. 앱바: 뒤로가기 + "크루 경기"
  2. 세그먼트 탭: 진행중 / 완료 / 취소
  3. 경기 카드:
     - 우리 크루 엠블럼(44px) + 크루명 / VS / 상대 크루 엠블럼+크루명
     - 경기 날짜·시간 + 핀명 + 종목 칩
     - 참가 크루원 아바타 미리보기 (최대 5개 + N명 더)
     - 상태 칩: 수락대기(노랑) / 경기확정(파랑) / 완료(초록/빨강) / 취소(회색)
     - 완료된 경기: 결과 표시(승/패, ELO 변동 +N/-N)
  4. 진행중 탭 상단: "경기 신청 가능" 안내 (크루장/부크루장에게만)
  5. 빈 상태: "아직 크루 경기가 없어요" + "첫 경기 신청하기" 버튼(크루장/부크루장만)

### 41. 크루 경기 신청
- **역할**: 상대 크루에게 크루 매칭 신청
- **진입 경로**: 크루 홈(34번) 하단 "크루 경기 신청" 버튼 / 크루 경기 목록(40번) FAB
- **접근 권한**: 크루장 / 부크루장
- **요소 (스텝 카드 형태)**:
  1. 앱바: 뒤로가기 + "크루 경기 신청"
  2. 스텝 1 — 상대 크루 선택:
     - 핀 선택 칩 (기본: 우리 크루 핀, 다른 핀도 선택 가능)
     - 상대 크루 카드 리스트 (같은 종목 기준 필터, 탭 가능)
     - 상대 크루 선택 시 강조 표시
  3. 스텝 2 — 참가 크루원 선택 **(소규모 종목만)**:
     - 단체 종목(축구/야구/농구): **이 스텝 건너뜀** — 참가 인원 선택 없이 바로 스텝 3으로
     - 소규모 종목: "참가 멤버 선택" 헤더 + 크루원 체크리스트 (아바타+닉네임, 체크박스)
       - 종목별 필수 인원 안내: "테니스 복식 2명을 선택해주세요" / "배드민턴 복식 2명을 선택해주세요" 등
       - 선택 인원이 종목 기준에 맞지 않으면 다음 버튼 비활성화
  4. 스텝 3 — 경기 일정 설정:
     - **단체 종목** (축구/야구/농구):
       - 캘린더 피커 (오늘 ~ 2주 뒤), **복수 날짜 선택 가능**
       - 선택한 날짜가 태그 칩으로 하단에 표시 (X 삭제 가능)
       - 또는 "요일 반복" 토글 ON → 요일 선택 칩 (월~일, 복수 선택) + 기간 설정 (이번 주/다음 주/2주간)
         - 예: "토, 일 / 2주간" → 해당하는 4개 날짜 자동 선택
       - 시간대 선택 (4열 그리드 8개 + 하루종일)
       - 안내 문구: "선택한 날짜 중 상대가 가능한 날에 매칭됩니다"
     - **소규모 종목** (테니스/배드민턴/골프/당구/볼링):
       - 날짜 선택 (오늘 ~ 내일, 단일 선택)
       - 시간대 선택 (4열 그리드 8개 + 하루종일)
     - 장소/메모 입력 (선택 입력, 최대 50자)
  5. "경기 신청하기" 버튼 (하단 고정)
- **복수 날짜 매칭 동작**:
  - 신청 시 선택한 모든 날짜가 상대에게 전달
  - 상대 수락 시 가능한 날짜 1개를 확정 선택
  - 매칭 상태: 날짜 미확정 시 "일정 조율중" → 확정 후 "경기 확정"
- **신청 완료 후**: 상대 크루장/부크루장에게 푸시 알림 발송 + 토스트 "경기 신청을 보냈어요!"

### 42. 크루 경기 수락 (전체화면 알림형)

- **요소 (위→아래)**:

  2. 상대 크루 정보:

  4. 상대 크루 참가 예정 멤버 **(소규모 종목만 표시)**: 아바타 가로 스크롤 + 닉네임 (N명)

     - 단체 종목은 이 섹션 숨김
  6. "거절"(아웃라인 빨강) / "수락"(오렌지) 2버튼
     - 복수 날짜 매칭: 날짜 미선택 시 수락 버튼 비활성화

- **요소**:
  1. 핀돌 로고 (중앙 상단, 64px)
  2. "업데이트가 필요합니다" (22pt Bold)
  3. "새로운 기능과 버그 수정이 포함된\n최신 버전으로 업데이트해 주세요" (14pt, #9CA3AF)
  4. 현재 버전 / 최신 버전 표시 (예: "현재 1.0.0 → 최신 1.2.0")
  5. "업데이트하기" 버튼 (오렌지, 풀사이즈) → 앱스토어/플레이스토어 이동
- **동작**: 업데이트 버튼 외 모든 인터랙션 차단, 뒤로가기 불가

### 58. 점검 사전 공지 배너 (홈 화면 상단)

### 서버 점검 시스템 상세

#### API 설계: `/app/config`
앱 실행 시 호출하는 설정 API로, 점검/업데이트 정보를 한 번에 반환:

```json
{
  "maintenance": {
    "enabled": false,
    "startAt": "2026-04-11T02:00:00+09:00",
    "endAt": "2026-04-11T04:00:00+09:00",
    "reason": "더 나은 서비스를 위해 점검 중이에요",
    "noticeBeforeHours": 3
  },
  "forceUpdate": {
    "minVersion": "1.2.0",
    "storeUrl": {
      "ios": "https://apps.apple.com/app/id...",
      "android": "https://play.google.com/store/apps/details?id=..."
    }
  }
}
```

#### 매칭 차단 로직

> **원칙: 점검으로 인해 매칭이 취소되는 일은 절대 없어야 한다.**
> 매칭 생성 단계에서 점검 시간대를 사전에 차단하여, 이미 잡힌 매칭이 점검 때문에 취소되는 상황을 원천 방지한다.

| 시점 | 동작 |
|------|------|
| **매칭 생성 — 시간 슬롯 선택** | 점검 시간과 겹치는 시간 슬롯 비활성화 (회색 처리) + 슬롯 하단 "점검 예정" 라벨 표시 |
| **매칭 생성 — "하루종일" 옵션** | 해당 날짜에 점검이 예약되어 있으면 "하루종일" 버튼 비활성화 + 툴팁 "점검 예정 시간이 포함되어 하루종일 매칭이 불가합니다" |
| **즉시 매칭 생성** | 점검 시작까지 남은 시간이 매칭 유효 시간보다 짧으면 생성 차단 + "점검 예정으로 매칭을 생성할 수 없습니다" 안내 |
| **점검 중** | 모든 API 요청 → 503 + 점검 안내 JSON 반환 (이미 잡힌 매칭은 점검 시간대에 걸리지 않으므로 영향 없음) |
| **점검 종료 후** | 앱이 자동으로 점검 해제 감지 → 정상 화면 복귀 |

#### 푸시 알림

| 시점 | Type | title | body |
|------|------|-------|------|
| 점검 N시간 전 | `MAINTENANCE_NOTICE` | 서버 점검 안내 | {시작시간}~{종료시간} 서버 점검이 예정되어 있습니다. |
| 점검 종료 | `MAINTENANCE_COMPLETE` | 점검 완료 | 서버 점검이 완료되었습니다. 다시 매칭을 시작해 보세요! |

#### 어드민 제어

| 기능 | 설명 |
|------|------|
| 점검 예약 | 시작/종료 시간 + 사유 입력 → 스케줄 등록 |
| 즉시 점검 | 긴급 점검 모드 ON (즉시 적용) |
| 점검 해제 | 예정보다 일찍 종료 시 수동 해제 |
| 사전 공지 시간 | 점검 몇 시간 전부터 배너/푸시 발송할지 설정 (기본 3시간) |
| 점검 이력 | 과거 점검 기록 조회 (시작/종료/사유) |

아래는 유저가 명시하지 않았지만 기획 확정 전 반드시 결정이 필요한 항목들입니다. 각 항목에 추천 방향을 병기합니다.

### A. 종목별 크루 규칙

#### 종목 분류

| 분류 | 종목 | 크루 최대 인원 | 경기 참가 인원 선택 | 매칭 날짜 범위 | 복수 날짜 매칭 |
|------|------|---------------|-------------------|--------------|-------------|
| **단체 종목** | 축구, 야구, 농구 | 무제한 | 선택 안 함 (전원 참가) | 오늘 ~ 2주 뒤 | O (요일 묶음 가능) |
| **소규모 종목** | 테니스, 배드민턴, 골프, 당구, 볼링 | 최대 3명 | 필수 선택 (종목별 인원) | 오늘 ~ 내일 | X |

#### 단체 종목 상세

- **크루 인원**: 가입 인원 제한 없음 (100명 이상 가능)
- **경기 참가**: 크루원 선택 절차 없음 — 경기 당일 누구든 참가 가능
- **매칭 날짜**: 최대 2주 뒤까지 선택 가능
- **복수 날짜 매칭**: "토, 일" / "월, 수, 금" 등 여러 날짜를 묶어서 한 번에 매칭 신청 가능
  - 상대 크루가 묶인 날짜 중 하나라도 가능하면 매칭 성사
  - 수락 시 확정 날짜 1개를 선택

#### 소규모 종목 상세

- **크루 인원**: 최대 3명까지만 가입 가능
- **경기 참가**: 종목별 경기 인원을 반드시 선택
  - 테니스: 2명 선택 (복식)
  - 배드민턴: 2명 선택 (복식)
  - 골프: 2~3명 선택
  - 당구: 1~2명 선택
  - 볼링: 1~3명 선택
- **매칭 날짜**: 개인 매칭과 동일 (오늘~내일)
- **복수 날짜 매칭**: 불가

#### 구성 및 가입

| 항목 | 규칙 |
|------|------|
| 개인 최대 크루 가입 수 | 최대 3개 크루 동시 가입 허용 (종목이 다른 경우 무제한, 같은 종목은 핀별 1개 크루) |

### E. 탭바 구성 — 4탭 유지 + 핀 지도 모드 토글

- **4탭 유지** (홈/핀/매칭/마이) — 탭 추가 없이 기존 구조 유지
- 크루 기능은 **핀 지도(탭 2) 우측 상단 모드 토글**로 통합
  - `개인` 모드: 기존 1:1 매칭 종목 마커 + 개인 랭킹
  - `크루` 모드: 크루 종목 마커 + 크루 목록/랭킹 + 크루 매칭
- 모드별 종목 분류 예시:
  - **개인 중심**: 골프, 당구, 테니스, 탁구, 배드민턴, 볼링, 가위바위보, 팔씨름
  - **크루 중심**: 축구, 농구, 야구 (+ 개인 종목도 크루로 가능)
- 장점: 5탭으로 늘리지 않아 UI 깔끔, 지도 기반 탐색이라 위치 맥락 자연스러움
- 내 크루 관리는 마이 탭(22번) 메뉴 "내 크루"로 진입

### G. 미결 결정 필요 항목 (기획 확정 전 논의 필요)

7. **크루 경기 신청 시 ELO 차이 제한**: 개인 매칭처럼 크루 ELO 차이가 너무 크면 매칭 제한 여부
