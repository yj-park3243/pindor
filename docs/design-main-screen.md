> 작성일: 2026-04-04 / 업데이트: 2026-04-09

## 2. 섹션별 상세

### 2.1 상단 바
- 앱 로고 "핀돌" (primaryColor)
- 알림 아이콘 (읽지 않은 알림 있으면 빨간 점 뱃지)
- 설정 아이콘 미구현 (추후 추가 예정)

### 2.2 매칭 수락 대기 배너 (_PendingAcceptBanner)
- `PENDING_ACCEPT` 상태 매칭이 있을 때만 상단에 표시
- 탭하면 매칭 수락 화면으로 이동

### 2.3 공지사항 배너 (_PinnedNoticeBanner)
- 핀 고정된 공지사항이 있을 때만 표시
- 없으면 섹션 숨김

### 2.4 매칭 CTA 버튼
- "오늘 대결 나가고 싶다!" — 탭 시 핀맵(`/map`) 화면으로 이동
- 핀맵에서 핀 선택 → 종목/날짜/시간 → 매칭 요청 흐름

### 2.5 내 핀 상태 카드 (_MyPinStatusCard)
- 마이페이지에서 설정한 `selectedPin` + `sportPreference` 기반
- **핀 미설정** 시: "내 핀을 설정해주세요." 표시
- **핀 설정됨**:
  - 핀 이름 + 선택 종목 표시
  - 통계 3열: 순위(미집계) / 점수(또는 "배치중") / 활동 중 인원
  - 배치 중이면 진행 상황 배지 표시 (`배치 중 N/5`)
  - 게시판 보기 버튼 → 해당 핀 게시판 이동
- 스포츠 프로필 없으면 종목 통계 `-` 표시

### 2.6 최근 전적 (_RecentMatchHistory)
- 최근 완료된 매칭 최대 5개 기반으로 전적 표시
- `GET /matches?status=COMPLETED&limit=5`

### 2.7 게시판 최신글 (_PinBoardPreview)
- 선택된 핀의 최신 게시글 최대 3개
- 각 게시글: 제목, 댓글/좋아요 수, 작성 시간(timeago)
- `GET /pins/:pinId/posts`

### 2.8 하단 탭바
| 탭 | 아이콘 | 기능 |
|----|--------|------|
| 홈 | house | 현재 화면 (메인) |
| 핀맵 | location_on | 지도 뷰 — 핀 탐색 + 매칭 요청 |
| 매칭 | flash_on | 매칭 요청/진행 관리 |
| 채팅 | chat_bubble | 채팅룸 목록 |
| 마이 | person | 내 프로필, 전적, 설정 |

## 3. 데이터 의존성

| 섹션 | Provider / API |
|------|----------------|
| 알림 뱃지 | `unreadNotificationCountProvider` |
| 매칭 수락 배너 | `pendingAcceptMatchesProvider` |
| 공지 배너 | `pinnedNoticeProvider` |
| 내 핀 | `selectedPinProvider` (SharedPreferences 기반) |
| 내 스포츠 프로필 | `currentUserProvider` → `sportsProfiles` |
| 선택 종목 | `sportPreferenceProvider` |
| 최근 전적 | `recentCompletedMatchesProvider` — `GET /matches?status=COMPLETED&limit=5` |
| 게시판 최신글 | `pinLatestPostsProvider(pinId)` — `GET /pins/:pinId/posts` (3개 take) |

## 4. 상태별 분기

| 상태 | 메인 화면 표시 |
|------|---------------|
| 핀 미설정 | 내 핀 카드에 "내 핀을 설정해주세요." |
| 핀 설정 + 스포츠 프로필 없음 | 점수/순위 `-` 표시 |
| 배치전 진행 중 | 점수 "배치중" + 진행 배지 (`N/5`) |
| PENDING_ACCEPT 매칭 있음 | 상단 수락 대기 배너 표시 |
| 공지사항 있음 | 공지 배너 표시 |
