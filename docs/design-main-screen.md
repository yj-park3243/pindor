## 2. 섹션별 상세

### 2.1 상단 바
- 앱 로고 "Spots"
- 알림 아이콘 (읽지 않은 알림 수 뱃지)
- 설정/프로필 아이콘

### 2.2 내 핀 영역
- **즐겨찾기 핀** 중 주 핀(primary)을 표시
- 핀 이름, 지원 종목 아이콘, 현재 활성 유저 수
- 탭하면 핀 상세 페이지 (게시판, 팀, 랭킹)
- 핀 전환: 좌우 스와이프로 즐겨찾기 핀 간 전환
- 핀 미등록 시: "핀을 등록하세요! 주변 핀 찾기 →"

### 2.3 매칭 CTA
- **빠른 매칭**: 현재 선택된 핀에서 즉시 매칭
  - 탭 → 종목 선택 → 해당 핀 매칭 풀에 등록
- **예약 매칭**: 핀 + 날짜/시간 지정
  - 탭 → 핀 선택(즐겨찾기 중) → 종목/날짜/시간 → 매칭 요청

### 2.4 내 등급 카드
- 현재 대표 종목의 티어 + 세부단계
- 프로그레스바 (다음 세부단계까지)
- "N점 더 필요" 텍스트
- 최근 전적 (5전 N승 N패)
- 여러 종목 등록 시 스와이프로 종목 전환

### 2.5 진행 중 매칭/게임
- 현재 진행 중인 매칭이나 게임이 있으면 카드로 표시
- 상태별 CTA:
  - `PENDING_ACCEPT`: [수락] [거절]
  - `CHAT`: [채팅]
  - `CONFIRMED`: [결과 입력]
  - `COMPLETED`: [리뷰 작성]
- 없으면 이 섹션 숨김

### 2.6 커뮤니티 피드
- 현재 핀의 종목별 인기 게시글
- 최신 + 인기순 혼합
- 각 게시글: 종목 아이콘, 제목, 댓글/좋아요 수, 시간
- "더 보기" → 핀 게시판 전체 보기

### 2.7 하단 탭바
| 탭 | 아이콘 | 기능 |
|----|--------|------|
| 홈 | 🏠 | 현재 화면 (메인) |
| 핀맵 | 📍 | 지도 뷰 — 주변 핀 탐색 |
| 매칭 | ⚡ | 매칭 요청/진행 관리 |
| 채팅 | 💬 | 채팅룸 목록 |
| 마이 | 👤 | 내 프로필, 전적, 팀, 설정 |

## 3. 데이터 의존성

| 섹션 | API 호출 |
|------|---------|
| 내 핀 | `GET /users/me` → userPins (즐겨찾기 핀 목록) |
| 내 등급 | `GET /users/me` + `GET /sports-profiles` |
| 진행 중 | `GET /matches?status=PENDING_ACCEPT,CHAT,CONFIRMED` |
| 커뮤니티 | `GET /pins/:pinId/posts?limit=5&sort=popular` |
| 알림 뱃지 | `GET /notifications?unreadOnly=true&limit=1` (count만) |

## 4. 상태별 분기

| 상태 | 메인 화면 표시 |
|------|---------------|
| 신규 유저 (종목 미등록) | 온보딩 카드: "종목을 선택하세요" |
| 종목 등록 완료 | 정상 메인 화면 |
| 매칭 대기 중 | 상단에 "매칭 대기 중..." 배너 |
| 게임 결과 미입력 | 진행 중 섹션에 강조 표시 |
| 정지 상태 | 정지 안내 오버레이 |
