계정 및 데이터 삭제 요청

서비스명: 핀돌  |  운영: 디자인비 (박용주)
핀돌 서비스의 계정 삭제를 요청하시면 회원 정보 및 관련 데이터가 영구적으로 삭제됩니다. 삭제가 완료되면 동일한 계정으로 다시 로그인하거나 데이터를 복구할 수 없습니다.

삭제되는 데이터

계정 삭제 요청 시 다음 데이터가 영구적으로 삭제됩니다.

  • 계정 정보: 이메일, 닉네임, 프로필 사진, 성별, 생년월일
  • 소셜 로그인 연동 정보: Apple, Google, Kakao 연동 데이터
  • 매칭 및 경기 기록: 매칭 요청, 매칭 결과, 경기 기록, 점수 이력
  • 채팅 기록: 모든 채팅 메시지 및 채팅방 정보
  • 커뮤니티 활동: 게시글, 댓글
  • 스포츠 프로필: 종목별 점수, 티어, 전적
  • 알림 내역: 모든 푸시 알림 및 앱 내 알림 기록

삭제 처리 절차

단계 내용 소요 기간
1. 요청 아래 이메일로 계정 삭제 요청 (가입 이메일 또는 닉네임 기재) -
2. 본인 확인 가입 시 사용한 이메일 또는 소셜 계정 정보로 본인 확인 1~2 영업일
3. 삭제 처리 계정 비활성화 후 30일 이내 모든 데이터 영구 삭제 최대 30일
4. 완료 안내 삭제 완료 후 이메일로 결과 안내 -

삭제 후 보관되는 데이터

관계 법령에 따라 아래 데이터는 일정 기간 보관 후 파기됩니다.

  • 계약 또는 청약 철회 기록: 5년 (전자상거래법)
  • 소비자 불만 또는 분쟁 처리 기록: 3년 (전자상거래법)
  • 접속 로그: 3개월 (통신비밀보호법)

위 법정 보존 기간이 경과하면 해당 데이터도 완전히 파기됩니다.
